# Video-Editor ✨  
Mescla NodeJs + Python para download do video, extrair o audio e separar vozes dos instrumentos  

## Pré requisitos  
- Python 3.8.20
- NodeJs v18.12.1
- FFMPEG 
- Spleeter    
- demucs
- sox

## Dependencias Python
```pip install pytube```

## Iniciar
- Dentro do arquivo youtube.py substituir ```inserir_url```
- Dentro do diretório root executar ```npm start```
